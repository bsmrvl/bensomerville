Chartree - Growing Trees of Character (Not Exactly Data Science)

Ben Somerville - Nov 17, 2020

For me, learning how to package Python programs and make them available online was a big breakthrough. Oh to add real projects to the real world!—empower madmen coders from all corners with my own built tools! So while my classmates put together basic done-a-thousand-times helper functions in order to grasp the packaging concepts, I poured a glass of cold coffee and typed up something extraordinarily useful.









Glorious!

Unfortunately this is just a gif. But if you happen to have a Python environment on your computer, you can very easily install the package and mess around with the wealth of parameters. Even the default settings grow a unique tree every time. And you can use whatever characters you like, for both tree and background.

Head over to Github for the installation and basic usage. If you don't have a Python environment, you can start by going to Google Colab, starting a new notebook, and running the command !pip install chartree (the "!" is necessary in Colab). You'll see a new code cell appear, where you can follow the instructions from Github. Enjoy!
